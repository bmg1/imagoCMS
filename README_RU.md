# imagoCMS

[English](README.md) | [Русский](README_RU.md) | [Latviešu](README_LV.md)

imagoCMS - быстрая, простая и безопасная система управления содержимым, способная расширятся как горизонтально так и вертикально, содержит такие модули как:
* Языки - с автоматическим определением и сохранением ранее открытого
* Внешнее API - подходит для связи с мобильными приложениями и другими системами
* Шаблонизатор WEBSUN - и может использовать любой удобный для вас
* Контролируемое кеширование - на уровне шаблона, данных, базы
* Удобную и настраиваемую систему управления
* Человекопонятные URL, разного уровня и модулей
* Встроенная система меню и модулей
* SEO оптимизация - sitemaps, robots.txt, favicon.ico
* White Label
* Систему deploy и тестирования

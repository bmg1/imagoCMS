# imagoCMS

[English](README.md) | [Русский](README_RU.md) | [Latviešu](README_LV.md)

imagoCMS — ātra, vienkārša un droša satura pārvaldības sistēma, kas spēj mērogoties gan horizontāli, gan vertikāli. Tā ietver šādus moduļus:
* Valodas — ar automātisku noteikšanu un iepriekš izmantotās valodas saglabāšanu
* Ārējais API — piemērots saziņai ar mobilajām lietotnēm un citām sistēmām
* WEBSUN veidņu dzinējs — ar iespēju izmantot jebkuru citu jums ērtu veidņu sistēmu
* Kontrolēta kešošana — veidņu, datu un datubāzes līmenī
* Ērta un pielāgojama administrēšanas sistēma
* Cilvēcīgi saprotami URL — dažādu līmeņu un moduļu maršrutēšanai
* Iebūvēta izvēlņu un moduļu sistēma
* SEO optimizācija — sitemaps, robots.txt, favicon.ico
* White Label
* Deploy un testēšanas sistēma

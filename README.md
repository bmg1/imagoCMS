# imagoCMS

[English](README.md) | [Русский](README_RU.md) | [Latviešu](README_LV.md)

imagoCMS is a fast, simple, and secure content management system designed to scale both horizontally and vertically. It includes the following modules and capabilities:
* Languages — automatic language detection with persistent user preference storage
* External API — ideal for integration with mobile applications and third-party systems
* WEBSUN templating engine — with the flexibility to use any template engine you prefer
* Controlled caching — at the template, data, and database levels
* Flexible admin panel — convenient, customizable content and system management
* Human-readable URLs — multi-level, module-aware routing
* Built-in menus and modules system — structured, extensible architecture
* SEO optimization — automatic sitemaps, robots.txt, favicon.ico
* White Label support — fully brandable
* Deployment and testing system — integrated tools for CI/CD and environment control
